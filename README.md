## 🚀 Project Description
<p>The 3D Animation was developed through the <a href="https://www.youtube.com/watch?v=XK7T3mY1V-w&ab_channel=DevEd">Dev Ed YouTube Channel</a>. 

It was used 3 kinds of EventListener ('mousemove', 'mouseenter', 'mouseleave'), Transition and, Transform (TranslateZ for the 3D aspect).

I hope it can be used in e-commerce to give a new customer experience for the e-shoppers</p>
